VINEYARD SOUNDS - Sounds Mans Shop store Website

A modern, responsive e-commerce website for VINEYARD SOUNDS, a premium music instruments store located in Eldoret, Kenya.

🎵 About

VINEYARD SOUNDS is a complete music instruments store website featuring a beautiful home page, a full shop with filtering and search, and a contact page with form and map. Built with clean HTML, CSS, and JavaScript — no frameworks or build tools required.

📂 Project Structure

├── contact.html        # Contact page (form, location info, map, FAQ)
├── css/

├── js/

🚀 How to Put This Website on GitHub (Step-by-Step)

Step 1: Create a GitHub Account

If you don't have one, sign up at github.com.

Step 2: Create a New Repository

- Click the "+" icon in the top right corner and select "New repository"

- Name it vineyard-sounds (or any name you prefer)

- Set it to Public

- Check "Add a README file"

- Click "Create repository"

Step 3: Upload Your Website Files

Option A — Upload via GitHub Web Interface:

- In your new repository, click "Add file" → "Upload files"

- Drag and drop all your files: index.html, shop.html, contact.html, and the css/ and js/ folders

- Write a commit message like "Initial website upload"

- Click "Commit changes"

Option B — Upload via Git Command Line:

# Navigate to your website folder
cd vineyard-sounds

# Initialize git
git init

# Add all files
git add .

# Commit
git commit -m "Initial website for VINEYARD SOUNDS"

# Add your GitHub repository as the remote (replace YOUR-USERNAME)
git remote add origin https://github.com/YOUR-USERNAME/vineyard-sounds.git

# Push to GitHub
git branch -M main
git push -u origin main

Step 4: Enable GitHub Pages (This makes your website LIVE on the internet!)

- Go to your repository on GitHub

- Click the "Settings" tab

- In the left sidebar, click "Pages"

- Under "Source", select the "main" branch

- Under "Folder", select "/ (root)"

- Click "Save"

Step 5: Access Your Live Website

- After a few minutes, GitHub will give you a URL like: https://YOUR-USERNAME.github.io/vineyard-sounds/

- This is your website, live on the internet for everyone to visit!
